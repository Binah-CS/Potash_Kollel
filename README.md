# Shutafut Static Website

This workspace now contains a plain static RTL website for "שותפות קהל שערי איש וכולל ארץ ישראל".

## Project structure

- `index.html` — דף הבית
- `about.html` — אודות הכולל
- `partnership.html` — על השותפות
- `donate.html` — דף תרומה
- `kvitel.html` — קוויטלעך וישועות
- `contact.html` — צור קשר
- `css/styles.css` — כל העיצוב המשותף לאתר
- `main.js` — אינטראקציות מינימליות ו-modal
- `assets/images/` — תמונות ודפי רקע
- `assets/videos/` — סרטון הכולל
- `assets/fonts/` — קבצי הגופנים המקומיים

## Usage

1. פתח את `index.html` או כל דף אחר בדפדפן כדי לראות את האתר.
2. החלף את התמונות בתיקיית `assets/images/` לפי הצורך.
3. החלף את הסרטון ב-`assets/videos/` אם תרצה סרטון מותאם.
5. ניתן לערוך את הצבעים, הפונטים והעיצוב ב-`css/styles.css`.
5. ניתן לשנות את ההתנהגות הקלה ב-`main.js`.

## Notes

- אין כאן שום build process או framework — רק HTML, CSS ו-JS פשוט.
- העיצוב מבוסס על פונטים מקומיים ב-`assets/fonts/`.
- הקישורים בדפים מפנים לדף HTML statically.
